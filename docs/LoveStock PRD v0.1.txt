LoveStock PRD v0.1
产品简介
用股票交易所的视觉语言，幽默地给用户的"恋爱市场价值"估值，生成一张可分享的个人股票卡片，导流到主配对产品 Blink。
MVP 目标：跑通完整流程，验证传播性。不追求功能完整，追求体验完整。

用户流程（从入口到退出）
Bot 发送入口消息
    ↓
用户点击「Open LoveStock」按钮
    ↓
Mini App 打开 → 欢迎页（2秒动画）
    ↓
答题页（8题，逐题显示）
    ↓
「正在计算估值...」加载页（2-3秒，假装计算）
    ↓
结果页：交易所大屏风格卡片
    ↓
弹出假通知：「你的股价已上市，X人正在关注你」
    ↓
通知消失后显示底部按钮：「找到能配对你股价的人 →」
    ↓
跳转 Blink bot

8道题设计
前5题搞笑，后3题性格向，最后综合出标签。
Q1 你上一次被人表白是什么时候？

A：最近一个月 (+20% 市值)
B：一年内 (+10%)
C：记不清了 (-5%)
D：从来没有 (-15%，但附加「稀缺资产」标签)

Q2 失恋后你一般多久恢复？

A：一周内，我是职业选手 (+15%，「高流动性资产」)
B：一个月左右 (+5%)
C：要好几个月 (-5%)
D：还没恢复过 (-10%，但附加「长期持有价值」标签)

Q3 你在关系里更像？

A：主动追人的那个 (+10%)
B：被追的那个 (+15%)
C：互相暗示然后谁都不说 (0%，「信息不对称资产」)
D：随缘，缘分到了自然在一起 (-5%)

Q4 你发消息一般多久回？

A：秒回 (-10%，「过度供给」)
B：几分钟内 (+10%)
C：想起来就回 (+5%)
D：我经常忘回 (-5%，但附加「神秘溢价」标签)

Q5 前任对你的评价可能是？

A：太粘人了 (-10%)
B：太冷淡了 (+5%，「稀缺资产」)
C：很好，是我配不上 (+20%)
D：我没有前任 (随机±15%，「未上市原始股」)

Q6（性格向）你理想的周末是？

A：和人出去玩，越热闹越好 → 外向型
B：和一两个好朋友安静待着 → 稳健型
C：自己在家，偶尔出门 → 独立型
D：完全随心情 → 波动型

Q7（性格向）关系里你最在意的是？

A：对方懂我，有共鸣 → 情感型
B：稳定感和安全感 → 稳健型
C：保持各自的空间 → 独立型
D：一起成长进步 → 成长型

Q8（性格向）你觉得自己最大的优点是？

A：有趣，在哪里都是气氛组
B：靠谱，说到做到
C：独立，不依赖别人
D：深情，认准了就是你


股票标签系统（结果类型）
根据 Q6/Q7/Q8 的组合，输出4种主标签：
标签英文人设蓝筹股Blue Chip稳定、靠谱、长期价值高成长股Growth Stock潜力大、有点不确定、值得押注概念股Concept Stock有趣、话题性强、但捉摸不定防御股Defensive Stock独立、不受外界影响、低波动

结果页内容结构
┌─────────────────────────────────┐
│  🏛️ LOVESTOCK EXCHANGE          │
│  NYSE • LIVE                    │
├─────────────────────────────────┤
│  [@username]                    │
│  TICKER: [3-4字母代号]           │
│  ──────────────────             │
│  CURRENT PRICE  $[估值]         │
│  ▲ +[涨幅]%  TODAY             │
│  ──────────────────             │
│  TYPE:  [Blue Chip / Growth...] │
│  GRADE: [A+ 到 C-]              │
│  SPECIAL TAG: [附加标签]         │
│  ──────────────────             │
│  [K线走势图，随机生成但看起来合理] │
│  ──────────────────             │
│  「一句话点评」                  │
└─────────────────────────────────┘
Ticker 规则：取用户名前3-4个字母大写，没有用户名就随机生成。
估值范围：$10 - $999，故意不给整数，比如 $347 或 $89。

假通知 + 导流
结果页出现3秒后，顶部滑入一条假通知：
📈 Market Alert
847 investors are watching your stock
数字随机生成（500-1200之间），停留4秒后滑出。
通知消失后底部按钮出现：
[ Find your perfect market match on Blink → ]
点击跳转 Blink bot 链接。

边缘情况
情况处理用户没有 Telegram 用户名Ticker 用随机4字母，显示名用 first name用户中途退出重进重新开始，不保存进度（MVP不做存档）网络断开答题全在前端完成，不需要网络，结果页再请求后端用户快速连点选项选中后禁用按钮300ms再跳下一题Telegram initData 验证失败显示通用错误页，不暴露原因

估值范围：$1 - $9,999，保留小数点后两位，比如 $4,721.38
一句话点评：OpenRouter 实时生成，模型用 google/gemini-2.0-flash-001（够快够便宜）
K线图：根据答案影响走势（激进答案→波动大，稳健答案→走势平缓）
